## 2D-path-planning
**二维路径规划——A星（A star）**

**_1.程序介绍_**：

**Astar.py**：算法主程序。

**Bidirectional_a_star.py**：双向A星算法主程序。

**env.py**：环境设置。

**plotting.py**：画图和动态显示轨迹。

**twocross.py**：判断碰撞。
